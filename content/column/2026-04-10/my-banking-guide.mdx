**この記事の日本円換算は、1MYR≒32.5円で計算しています（2026年3月時点）。為替は変動するので、現地通貨（MYR）の金額を基準にしてください。**

マレーシアに移住する日本人が最初に直面する手続きのひとつが銀行口座の開設だ。

クアラルンプールでは就労・長期居住の外国人向けに銀行口座の開設は比較的開かれているが、Employment Pass（就労パス）とMM2H（マレーシア・マイ・セカンドホーム）では必要書類が異なり、手続きの経路も変わってくる。どちらのビザを持っているかを確認してから銀行に向かうのが時間の節約になる。

## ビザ別の口座開設条件

| ビザ種別 | 口座開設 | 必要な追加書類 |
|---|---|---|
| **Employment Pass（就労パス）** | 可能 | 雇用契約書・在職証明書 |
| **MM2H（マレーシア・マイ・セカンドホーム）** | 可能 | MM2H承認レター・定期預金証明（場合によっては） |
| **DE Rantau（デジタルノマドビザ）** | 可能（条件あり） | フリーランス契約書・収入証明 |
| **学生ビザ（Student Pass）** | 可能（在学証明が必要） | 在学証明書 |
| **観光ビザ（ノービザ含む）** | 原則不可 | — |

Employment Pass保持者はマレーシアで就労する外国人向けの一般的なビザで、雇用主が手続きをサポートしてくれることが多い。MM2Hは長期居住プログラムで、2021年のリニューアル以降は最低財産要件が引き上げられた（2024年時点では500万MYR相当の流動資産などの条件がある）が、取得後の銀行口座開設は比較的スムーズだ。

## 主要銀行の比較

| 銀行 | 特徴 | 外国人対応 | 最低預入額（MYR） |
|---|---|---|---|
| **Maybank（メイバンク）** | 国内最大手・ATM最多 | 英語対応・外国人実績豊富 | 0〜500（口座タイプによる） |
| **CIMB Bank** | ASEANで展開・アプリが優秀 | 英語対応 | 0〜1,000 |
| **Public Bank** | 手堅い・富裕層に人気 | 英語対応 | 0〜 |
| **RHB Bank** | 中堅・DuitNow対応 | 英語対応 | 0〜 |
| **Hong Leong Bank** | オンライン口座開設に積極的 | 英語対応 | 0〜 |

Employment Pass保持者に最も人気が高いのはMaybankだ。国内ATM数が最多で、どのモールに行っても必ず見つかる。モバイルアプリ「MAE（Maybank2u）」は英語対応しており、DuitNow・口座振込・海外送金がアプリで完結する。

CIMBはASEAN全域に展開する銀行で、タイ・シンガポール・インドネシアにも支店がある。他国との資金移動が多い場合はCIMBの「ASEAN transfer」機能が使いやすい。アプリ「CIMB Clicks」も英語UIで使いやすく、外国人の評価が高い。

MM2H保持者の間ではPublic Bankを選ぶケースも多い。外国人の資産管理に対応する商品が豊富で、定期預金やマルチカレンシー口座の選択肢が広い。

## 必要書類

### Employment Pass保持者

| 書類 | 備考 |
|---|---|
| パスポート原本 | Employment Passスタンプまたはシール付き |
| Employment Pass原本 | 入国管理局（JTK）発行 |
| 在職証明書（Employment Letter）| 英語・会社のレターヘッド付き |
| 住所証明 | 賃貸契約書・光熱費の請求書（銀行によって要否が異なる） |

### MM2H保持者

| 書類 | 備考 |
|---|---|
| パスポート原本 | MM2H長期ビザが押されているもの |
| MM2H承認レター | 観光局（Tourism Malaysia）発行 |
| 住所証明 | 賃貸契約書 or ユーティリティ請求書 |
| 資産証明（一部銀行） | 定期預金証明書・固定資産証明等 |

MM2Hで定期預金の条件（固定資産MYR 500,000相当など）を満たしている場合、銀行がその証明書を住所証明の代替として受け入れることがある。窓口で事前に確認するのが確実だ。

### DE Rantau（デジタルノマドビザ）保持者

DE Rantauは2022年に導入されたデジタルノマド向けビザ。Employment Passと異なり雇用主がいないため、以下の書類で対応する。

| 書類 | 備考 |
|---|---|
| パスポート原本 | DE Rantauビザスタンプ付き |
| フリーランス契約書または雇用証明 | 英語で発行されたもの |
| 収入証明 | 直近3〜6か月の銀行取引明細・Payoneer等の送金記録 |
| 住所証明 | コワーキングスペースの会員証・賃貸契約書 |

## 口座開設の手順

1. **支店の選択**: クアラルンプールのKLCC・モントキアラ・バンサーエリアの支店が外国人対応に慣れている
2. **書類を全て揃えてから訪問**: 書類が揃っていないと当日は完結しない。書類リストは各銀行の公式サイトで確認するか事前に電話で確認する
3. **初回入金**: Maybankは通常MYR 500〜を初回入金として求める（口座タイプによる）
4. **MAE / CIMB Clicksのアプリ設定**: 窓口でその場で設定する支店もある

## DuitNow——マレーシア版即時送金

DuitNowはマレーシア国立銀行（BNM）が推進する即時決済システムで、電話番号・国民ID番号・パスポート番号を使ってリアルタイム送金ができる。外国人はパスポート番号でDuitNowに登録できる。

2026年時点でQRコード払い（DuitNow QR）も普及しており、KLのほとんどの飲食店・コンビニ・モールで使える。MAEアプリにDuitNowを登録しておけば、現金不要でほぼ全ての支払いに対応できる。

## Touch 'n Go eWalletとの使い分け

タッチ・アンド・ゴー（TnG）eWalletはマレーシアで最も普及したデジタルウォレットだ。高速道路の通行料（ELite・PLUS等の有料道路）への支払いに使われるだけでなく、コンビニ・フードデリバリー・市場での支払いにも対応している。

外国人はパスポートで登録でき、銀行口座がなくてもコンビニで現金チャージして使える。ただし1日の送金上限・残高上限に制限がある（eKYC未完了の場合）。

| 決済手段 | 主な用途 |
|---|---|
| **DuitNow（MAEまたはCIMBから）** | 個人間送金・QRコード払い |
| **TnG eWallet** | 高速道路・コンビニ・フードデリバリー |
| **銀行デビットカード** | モール・ホテル・大型店舗 |

銀行口座が開いてからは、日常的な送金はDuitNow、高速道路や一部コンビニではTnGという使い分けがシンプルだ。

## 海外送金——マレーシア↔日本

### 銀行送金（SWIFT）

| 項目 | 目安 |
|---|---|
| 送金手数料（マレーシア側） | MYR 10〜25（約325〜813円）＋ コルレス手数料 |
| 為替マージン | 0.5〜2% |
| 着金日数 | 1〜3営業日 |

### Wise経由の送金

WiseはMYR→JPYの送金に対応している（2026年3月時点）。マレーシアの銀行口座からWiseに送金し、日本の口座に受け取る流れが利用できる。手数料の透明性が高く、銀行SWIFTの総コストと比較すると安くなるケースが多い。

### Maybank「Remittance」サービス

MAEアプリ内の送金機能からSWIFT送金が可能で、日本の主要銀行への送金に対応している。手数料はMYR 10〜25程度で、アプリ内で完結する。深夜でも送金手続き自体はできるが、銀行処理は翌営業日になる。

## 注意点

### 住所証明の入手タイミング

KLに到着したばかりの場合、光熱費の請求書（TNB電力の請求書等）はすぐに手に入らない。賃貸契約書を代替として提出するか、銀行に事前に「到着直後でも口座を開けるか」を電話確認してから訪問する。

Maybankはこのケースに慣れており、雇用証明書があれば住所証明の書式に柔軟に対応してくれることが多いとされる。

### 外国人は利子収入に税がかかる場合

マレーシアは外国人の銀行利子収入に源泉徴収が発生する場合がある。2026年時点の税率については最新の税務情報を確認してほしい。日本在住の税務申告との関係も含め、不安があれば現地の日系税理士に相談する選択肢もある。

## まとめ——口座開設の手順

1. **Employment Pass保持者**: Maybankかつモントキアラ・バンサーの支店で。書類を揃えてから訪問
2. **MM2H保持者**: Public BankかMaybankで。定期預金の証明書類を持参する
3. **DE Rantau保持者**: CIMBがDE Rantau保持者の対応実績がある（支店に事前確認推奨）
4. **MAEアプリを入れてDuitNowを設定**: KLの生活に必要なキャッシュレスの土台が整う

口座を開いた翌週には、DuitNowとTnG eWalletを使いこなせる状態になる。キャッシュレスの密度はシンガポールに次ぐレベルで、現金を持ち歩く場面が驚くほど減る。
