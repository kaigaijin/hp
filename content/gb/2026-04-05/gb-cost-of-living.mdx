**この記事の日本円換算は、1GBP≒191円で計算しています（2026年4月時点）。為替は変動するので、現地通貨（GBP）の金額を基準にしてください。**

ロンドンの生活費には覚悟が必要だ。

物価がじわじわ上がっていたのが2010年代だとすれば、2020年代前半のインフレでその上昇が加速した。2025年時点でも、食料品・光熱費・住居費のいずれも日本より明確に高い。駐在員として手当がついた状態でも生活費は高いと感じるし、現地採用ならロンドンで貯金するのはかなり難しいというのが実態だ。

ただ、選び方次第でコントロールできる部分もある。Zone・スーパーの選択・通信キャリアで月の出費は大きく変わる。

## 食費

### スーパーでの自炊

ロンドンのスーパーは価格帯がはっきり分かれている。

| チェーン | 特徴 | 1人1週間の食材費（GBP） |
|---|---|---|
| Aldi / Lidl | ディスカウント系。最安値 | 20〜35 |
| Sainsbury's / Tesco | 中価格帯。品揃え◎ | 35〜60 |
| Waitrose / M&S | 高価格帯。品質重視 | 60〜100 |

アジア系スーパー（Wing Yip、H Mart等）では日本食材が手に入る。醤油GBP 3〜5、豆腐 GBP 1.5〜3、精米1kg GBP 3〜6程度。

## 住居費

ロンドンの住居費はZone（地域区分）によって大きく変わる。Zone 1が中心部（ウェストエンド・シティ）、数字が増えるほど郊外になる。

### 1ベッドルーム（1人〜カップル）

| Zone | 月額目安（GBP） |
|---|---|
| Zone 1（中心部） | 2,200〜4,000 |
| Zone 2（Brixton / Hammersmith等） | 1,800〜3,000 |
| Zone 3〜4（Wimbledon / Croydon等） | 1,400〜2,500 |
| Zone 5〜6（郊外） | 1,200〜2,000 |

### 2〜3ベッドルーム（家族向け）

| Zone | 月額目安（GBP） |
|---|---|
| Zone 1〜2 | 3,000〜6,000 |
| Zone 3〜4 | 2,200〜4,000 |
| Zone 5〜6 | 1,800〜3,200 |

日本人在住者が比較的多いエリア: Richmond（Zone 4）、Wimbledon（Zone 3/4）、Acton（Zone 3）。日本語補習校へのアクセスや日系コミュニティの集まりやすさを重視する場合、西ロンドン方面が選ばれることが多い。

## 交通費

### Oyster / コンタクトレス

ロンドンの地下鉄（Underground）・バス・DLR等はOysterカードまたはコンタクトレス決済（クレカ・スマホ）で乗れる。1日の利用上限（Daily Cap）があり、乗り過ぎても上限以上は請求されない仕組み。

| 経路 | 定期代（月額・GBP） |
|---|---|
| Zone 1〜2 定期 | 約170 |
| Zone 1〜3 定期 | 約205 |
| Zone 1〜4 定期 | 約240 |
| Zone 1〜6 定期 | 約300 |

### バス

1回乗車 GBP 1.75（2025年現在）。1時間以内は乗り換えが無料（Hopper fare）。定期は地下鉄と共通のTfL定期で対応できる。

## IHS（医療利用料）の実質負担

就労ビザ申請時に支払うIHS（GBP 1,035/年）は、NHSを利用した分の実質的な医療費ともいえる。月換算で GBP 86程度（約16,400円）がすでにコストとして発生している計算だ。

一方でNHSは入院・手術・救急含めほぼ無料で使えるため、大きな医療費が発生した場合には元が取れる仕組みになっている。

## 月額生活費の目安

### 一人暮らし（Zone 3〜4想定）

| 項目 | 低め（GBP） | 高め（GBP） |
|---|---|---|
| 家賃（Zone 3〜4） | 1,400 | 2,200 |
| 食費（自炊中心） | 200 | 450 |
| 交通費（Zone 1〜3定期） | 205 | 205 |
| 通信費 | 15 | 35 |
| その他（日用品・外食・娯楽） | 200 | 600 |
| **合計** | **2,020** | **3,490** |

一人暮らしの生活費は GBP 2,500〜4,000/月（約48〜76万円）が現実的なレンジ。

### 家族4人（子ども2人）

| 想定 | 月額目安（GBP） |
|---|---|
| 家賃（Zone 3〜4、2BR） | 2,200〜3,500 |
| 食費 | 700〜1,200 |
| 交通費（大人2人定期） | 400 |
| 通信費 | 60〜100 |
| その他 | 600〜1,500 |
| **合計** | **3,960〜6,700** |

日本人補習校の授業料や子どもの習い事、保育費（全日保育は月 GBP 1,000〜2,000）を加えると、家族での生活費は GBP 6,000〜12,000/月になることもある。
