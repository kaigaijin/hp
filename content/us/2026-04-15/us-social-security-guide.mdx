アメリカに着いて最初にやるべきことのひとつがSSN（Social Security Number）の取得だ。日本のマイナンバーに相当するが、利用場面の広さは比較にならない。SSNなしでは銀行口座が開きにくく、クレジットカードも作れず、賃貸審査も通りにくい。就労もできない。アメリカ生活の土台になる番号だ。

## SSNとは何か

SSN（Social Security Number）は9桁の番号で、SSA（Social Security Administration / 社会保障局）が発行する。

本来は社会保障給付（年金・障害給付等）のための番号だが、実際には税務申告（IRS）・雇用・金融機関・医療・行政手続きのあらゆる場面で使われている。一度取得した番号は原則として一生変わらない。

SSNは表面上シンプルな番号だが、これが個人識別子として機能しているため、番号の盗用（Identity Theft）がアメリカで深刻な問題になっている背景もある。

## 申請方法

SSAオフィスで申請する。アメリカ国内に約1,200箇所のSSAオフィスがあり、ウォークインまたはオンラインで予約できる。

就労ビザの場合、一般的な手順:
1. 入国後10日程度待つ（入国情報がSSAのシステムに反映されるのに数日〜1週間かかる）
2. 近くのSSAオフィスに訪問
3. 必要書類を提出
4. 2〜4週間でカード（物理）が郵送される

**必要書類（一般的なケース）:**
- パスポート（原本）
- ビザ（原本）
- I-94（入国記録。CBP One等で確認）
- 雇用オファーレター（就労ビザの場合）
- 現地住所の証明（銀行明細・光熱費請求書等）

## 就労ビザ別の申請タイミング

ビザの種類によって、SSN申請ができるタイミングが異なる。

| ビザ種別 | SSN申請可否 |
|---|---|
| H-1B（専門職） | 可。入国後10日以降に申請 |
| L-1（企業内転勤） | 可 |
| O-1（特別技能） | 可 |
| J-1（交流訪問者） | 条件あり（就労許可証がある場合は可） |
| F-1（学生ビザ） | 就労許可（OPT・CPT）がある場合のみ可 |
| B-1/B-2（観光・商用） | 不可 |

F-1ビザで学生として在学中の場合、SSNは取得できず、代わりにITIN（後述）を使う。

## SSNがないと何が困るか

SSNなしで生活するのは「できないことが多くて不便」というレベルではなく、実際には生活インフラが機能しないに近い状態になる。

**銀行口座**: 多くの銀行でSSNが必要。一部の銀行（Chase、Bank of America等）ではパスポート・ITINで口座開設できるケースもあるが、サービスに制限がかかることがある。

**クレジットカード**: アメリカのクレジット審査にはSSNとクレジットヒストリー（信用履歴）が必要。移住直後は信用履歴がゼロなので、Secured Card（デポジット担保型）から始める人が多い。

**賃貸**: 家主や不動産会社がSSNによる信用調査（Credit Check）を要求するのが一般的。SSNなし・クレジットなしだと数ヶ月分の家賃前払いを求められることがある。

**携帯電話（ポストペイド）**: AT&T・Verizon・T-MobileなどのPostpaidプランの契約にSSNが必要。SSNがない間はプリペイドSIM（GoPhone・Mint Mobile等）で対応するのが現実的。

**就労**: 雇用主はForm I-9（雇用資格確認）でSSNの提示を求める義務がある。SSNなしでは合法的に雇用できない。

## ITINとの違い

ITIN（Individual Taxpayer Identification Number / 個人納税者識別番号）は、SSNを取得できない人（F-1学生・観光ビザ等）がIRSへの申告・納税のために使う番号だ。

| 項目 | SSN | ITIN |
|---|---|---|
| 発行機関 | SSA（社会保障局） | IRS（内国歳入庁） |
| 取得対象 | 就労ビザ・市民権・永住権 | SSN不要の就労ビザ以外 |
| 就労への使用 | 可 | 不可（就労資格は別に必要） |
| 主な用途 | 就労・銀行・税金・生活全般 | 税務申告のみ |

ITINはあくまで税務申告用で、「就労への使用不可」「銀行審査の通りにくさ」という制限がある。就労ビザが取れた時点でSSNに切り替えるのが基本の流れだ。

## 日米社会保障協定（二重払い回避）

日本とアメリカは社会保障協定を締結している（1984年署名、2005年改定）。

主な効果は二重払いの回避。アメリカで就労するとFICA（Social Security Tax + Medicare Tax）が給与から控除される。日本から短期派遣される場合、日本の社会保険を継続していれば、アメリカのFICAへの支払いを免除できる可能性がある。

また相互に加入期間を通算できるため、アメリカのSocial Security（老齢年金）受給資格（原則40クレジット = 10年分相当）に必要な期間を日米通算で満たすことができる。
