**この記事の日本円換算は、100KRW≒10.5円で計算しています（2026年3月時点）。為替は変動するので、現地通貨（KRW）の金額を基準にしてください。**

月給が東京とほぼ変わらない水準で、生活費も東京並みかそれ以上——。「韓国は物価が安い」というイメージを持って来ると、最初の家賃相場で驚く。

特に外国人が住みやすいエリアは集中している。江南（카남）・梨泰院（이태원）・龍山（용산）・弘大（홍대）周辺は、日本人・外国人駐在員の多い人気エリアで家賃が高い。

実際の数字で整理する。

## 食費

### コンビニ・屋台（편의점 / 포장마차）

韓国のコンビニは CU・GS25・세븐일레븐が3強。弁当・おにぎり・カップラーメン・サラダが充実している。

| 品目 | 価格 |
|---|---|
| 삼각김밥（おにぎり） | KRW 1,200〜2,000（約126〜210円） |
| 도시락（弁当） | KRW 3,500〜6,000（約368〜630円） |
| 컵라면（カップラーメン） | KRW 1,300〜1,800（約137〜189円） |
| 편의점 커피（コンビニコーヒー） | KRW 1,500〜2,000（約158〜210円） |

コンビニで昼食を済ませれば KRW 5,000〜8,000（約525〜840円）で収まる。

### ローカル食堂・韓食堂（식당）

백반（定食）・순대국밥（スンデクッパ）・된장찌개（テンジャンチゲ）などのローカル食堂は、1食 KRW 8,000〜15,000（約840〜1,575円）が相場。

| 品目 | 価格 |
|---|---|
| 백반（定食） | KRW 8,000〜10,000（約840〜1,050円） |
| 냉면（冷麺） | KRW 12,000〜18,000（約1,260〜1,890円） |
| 삼겹살（サムギョプサル、1인분） | KRW 14,000〜20,000（約1,470〜2,100円） |
| 치킨（フライドチキン、1羽） | KRW 18,000〜24,000（約1,890〜2,520円） |

ランチ時間帯は定食メニューが比較的安い。夜の居酒屋（호프）で삼겹살＋맥주（ビール）となると、1人 KRW 25,000〜40,000（約2,625〜4,200円）になる。

### 日本食

ソウルには日本食レストランが多い。梨泰院・明洞・弘大エリアに集中。

| ジャンル | 価格帯 |
|---|---|
| 라멘（ラーメン専門店） | KRW 13,000〜18,000（約1,365〜1,890円） |
| 스시 / 이자카야（寿司・居酒屋） | KRW 20,000〜50,000（約2,100〜5,250円） |
| 고급 일식（高級日本料理） | KRW 50,000〜150,000以上 |

「日本と同じ味」を求めると出費が嵩む。ローカル食と使い分けるのが現実的。

### 自炊

이마트（イーマート）・롯데마트（ロッテマート）・홈플러스（ホームプラス）が主要スーパー。

| 品目 | 価格 |
|---|---|
| 쌀 1kg（米） | KRW 3,000〜5,000（約315〜525円） |
| 달걀 10개（卵10個） | KRW 2,000〜3,500（約210〜368円） |
| 돼지고기 삼겹살 100g（豚バラ） | KRW 2,500〜4,000（約263〜420円） |
| 신선한 채소（野菜） | 品目により KRW 1,500〜5,000 |

日本食材（醤油・みりん・味噌等）は日系スーパーや한남동（漢南洞）周辺の輸入食材店で手に入るが、割高。

## 住居費

詳細は[韓国（ソウル）の住居・賃貸ガイド](/kr/kr-housing-guide)を参照。ここでは概要のみ。

### チョンセ vs ウォルセ

外国人の大多数はウォルセ（月払い家賃）を選ぶ。チョンセは保証金が物件価格の50〜80%に相当し、ワンルームでも数千万KRW規模になるため、短〜中期滞在では現実的でない。

### エリア別ウォルセ月額（ワンルーム、目安）

| エリア | 特徴 | 月家賃目安（KRW） |
|---|---|---|
| 강남（江南） | 高級住宅・商業エリア。外国人・駐在員多い | 1,200,000〜2,500,000 |
| 이태원（梨泰院） | 外国人コミュニティの中心。外食が充実 | 1,000,000〜2,000,000 |
| 용산（龍山） | 都心へのアクセス良好。日本人駐在員多い | 900,000〜1,800,000 |
| 홍대（弘大） | 若者・文化エリア。ビジネス往来にも | 700,000〜1,500,000 |
| 마포（麻浦） | 弘大周辺。比較的手頃 | 600,000〜1,200,000 |
| 노원（蘆原） | 郊外。通勤が長くなるが家賃は安い | 450,000〜800,000 |

ソウル近郊（경기도: 京畿道）まで範囲を広げれば、さらに安い物件が見つかる。

**保証金**: ウォルセの場合、保証金（보증금）は通常 KRW 5,000,000〜20,000,000（約52.5万〜210万円）程度。

### ファミリー向け（3LDK以上）

| エリア | 月家賃目安（KRW） |
|---|---|
| 강남・서초（江南・瑞草） | 3,000,000〜8,000,000以上 |
| 용산・이태원 | 2,500,000〜5,000,000 |
| 성동（城東）・마포 | 1,800,000〜3,500,000 |

日本人学校（瑞草区）に通う場合、서초구（瑞草区）・강남구（江南区）に住む家族が多い。

## 交通費

### 地下鉄・バス

ソウルの地下鉄は T-money カードで乗車。初乗り KRW 1,400（約147円）。路線によって若干異なる。

| 交通手段 | 料金目安 |
|---|---|
| 지하철（地下鉄、市内） | KRW 1,400〜2,000（距離加算制） |
| 시내버스（市内バス） | KRW 1,200〜1,500 |
| 광역버스（広域バス、郊外路線） | KRW 2,800〜3,200 |

定期券はなく、月の利用額が KRW 60,000 超えると乗換割引が適用される仕組み（60,000원 초과분의 20%割引）。

### タクシー

| 種別 | 初乗り | 特徴 |
|---|---|---|
| 일반택시（一般タクシー） | KRW 4,800（2km） | 深夜（23:00〜4:00）は20〜40%割増 |
| 모범택시（模範タクシー） | KRW 7,000 | 上位グレード。品質高め |
| KakaoT | 一般と同額〜 | アプリ配車。韓国語不要で使いやすい |

KakaoT（카카오T）はアプリで行き先を入力するだけで配車でき、日本語設定も可能。深夜はサージ料金が加わる。

**月の交通費目安**: 地下鉄・バス中心なら KRW 80,000〜120,000（約8,400〜12,600円）。

## 通信費

主要3キャリアは SKT（SK텔레콤）・KT・LG U+。MVNOも充実している。

| プラン種別 | 月額目安（KRW） | 特徴 |
|---|---|---|
| データ5GB程度 | 30,000〜45,000（約3,150〜4,725円） | ライトユーザー向け |
| データ無制限（速度制限あり） | 55,000〜70,000（約5,775〜7,350円） | 一般的な使い方 |
| 完全無制限（動画・テザリング） | 75,000〜90,000（約7,875〜9,450円） | ヘビーユーザー向け |

ARC 取得前でも契約できるプランがあるが、機種変更・機種持ち込みの場合は窓口での手続きが必要。

## 医療費

NHIS（国民健康保険）に加入済みなら、外来診察は3割負担が基本。

| 受診内容 | 自己負担目安（NHIS加入後） |
|---|---|
| 内科・一般外来（1次医療機関） | KRW 3,000〜8,000（約315〜840円） |
| 総合病院・2次医療機関 | KRW 10,000〜30,000（約1,050〜3,150円） |
| 薬局（処方薬） | KRW 3,000〜10,000（約315〜1,050円） |
| 歯科（保険適用の基本処置） | KRW 10,000〜20,000（約1,050〜2,100円） |

日本語対応クリニックはソウル市内に複数あるが、保険適用外・割増料金になる場合がある。

## 月額生活費の目安

### 一人暮らし

| 項目 | 月額目安（KRW） |
|---|---|
| 家賃（弘大・麻浦周辺ウォルセ） | 700,000〜1,200,000 |
| 食費（外食＋自炊） | 400,000〜700,000 |
| 交通費 | 80,000〜150,000 |
| 通信費 | 40,000〜80,000 |
| 光熱費 | 60,000〜150,000（冬は暖房費が上がる） |
| 雑費・娯楽 | 100,000〜300,000 |
| **合計** | **約1,380,000〜2,580,000（約14.5〜27.1万円）** |

### 家族4人（夫婦＋子ども2人、江南・瑞草エリア）

| 項目 | 月額目安（KRW） |
|---|---|
| 家賃（3〜4LDK） | 3,000,000〜6,000,000 |
| 食費 | 800,000〜1,500,000 |
| 子どもの教育費 | 500,000〜2,000,000（学校種別による） |
| 交通費 | 200,000〜400,000 |
| 通信費（家族分） | 100,000〜200,000 |
| 光熱費 | 100,000〜250,000 |
| 雑費・娯楽・衣料 | 200,000〜500,000 |
| **合計** | **約4,900,000〜10,850,000（約51.5〜114万円）** |

江南・瑞草エリアの家族生活費が高い理由は家賃と教育費。住む場所をずらすだけで大きく変わる。
