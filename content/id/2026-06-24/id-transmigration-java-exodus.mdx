「なぜカリマンタンの市場でジャワ料理が売っているんですか？」——インドネシアを旅した日本人が感じる素朴な疑問の背後に、半世紀にわたる国家プロジェクトが隠れている。

## トランスミグラシとは

トランスミグラシ（Transmigrasi）は「移住」を意味するインドネシア語だ。オランダ植民地時代に始まり、スハルト政権下（1967〜1998年）に大規模化した国家主導の人口移動計画だ。

目的は二つだった。一つは人口過密のジャワ・バリ・マドゥラからの「圧力緩和」。もう一つは、人口が少なかった「外島」（スマトラ・カリマンタン・スラウェシ・パプア等）の「開発」だ。

スハルト政権の35年間で、数百万人規模の家族が政府の斡旋によりジャワから外島に移住したとされる。移住者には土地と農具が提供され、自立的な農業コミュニティを形成することが期待された。

## 先住民との軋轢

問題は「外島に人が住んでいなかったわけではない」ことだ。カリマンタンにはダヤク族が、パプアにはパプア先住民が、それぞれの生活圏を持っていた。

移住者が与えられた土地の多くは、先住民の伝統的な利用地（焼き畑農地、狩猟地、聖地）と重なっていた。法的な所有権がなかった先住民の土地は「未利用地」として扱われ、移住者に割り当てられた。

衝突は繰り返された。1990年代末のカリマンタン暴動では、ダヤク族とマドゥラ島からの移住者の間で大規模な民族間衝突が起き、多数の死者が出た（報道ベース）。

## 政策の変容と現在

民主化後の1999年以降、大規模なトランスミグラシは縮小した。強制移住から自発的移住への転換が試みられ、先住民の土地権利への配慮も一定程度組み込まれるようになった。

しかしトランスミグラシの「遺産」は消えない。カリマンタン、スラウェシ、スマトラの各地に、ジャワ語や バリ語を話すコミュニティが点在している。文化的なモザイクは複雑になった一方、民族間の緊張が潜在する地域も残っている。

## 首都移転との連続性

ヌサンタラ（カリマンタン東部）への首都移転計画は、新たな形の人口移動を引き起こしている。政府職員、建設労働者、関連産業の人材が移入し、地元の土地価格が急騰している。

「開発のために人を動かす」という発想は変わっていない。形が変わっただけで、土地と人の複雑な関係は続いている。
