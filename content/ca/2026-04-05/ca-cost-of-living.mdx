**この記事の日本円換算は、1CAD≒110円で計算しています（2026年4月時点）。為替は変動するので、現地通貨（CAD）の金額を基準にしてください。**

カナダの物価は日本より高い——というのは知っていたつもりだった。でも実際に住み始めると、通信費の請求書を見て二度見する人が多い。

月 CAD 100〜120（約11,000〜13,200円）の携帯代。カナダは先進国の中でも通信費が最も高い国のひとつとして知られており、大手3社（Rogers / Bell / Telus）が市場を支配している。食費・住居費も決して安くない。

ただし、都市の選択で差はかなり出る。カルガリーはトロント・バンクーバーと比べて家賃が30〜40%安く、州税がない（アルバータ州は所得税はあるが州消費税なし）という優位性がある。

## 食費

### スーパーでの自炊

| チェーン | 特徴 | 1人1週間（CAD） |
|---|---|---|
| No Frills / Food Basics | ディスカウント系。最安値 | 40〜70 |
| Loblaw / Sobeys | 中価格帯。品揃え良い | 60〜100 |
| Whole Foods / Farm Boy | 高品質・オーガニック | 100〜180 |
| T&T Supermarket | アジア系食材が充実。日本食材も豊富 | 50〜90 |

T&T（Loblaw傘下のアジア系スーパー）はトロント・バンクーバー・カルガリー等に展開しており、日本食材（醤油・味噌・日本米）が比較的安価で手に入る。

### 外食

| 場面 | 価格目安（CAD） |
|---|---|
| カジュアルランチ（フードコート・テイクアウト） | 12〜22 |
| カフェ（ティム・ホートンズのコーヒー） | 2〜4 |
| スターバックス（コーヒー） | 5〜8 |
| カジュアルレストランのディナー（1人） | 25〜50 |

### 日本食レストラン

バンクーバーは北米有数の日本食の街として知られており、品質の高い日本食レストランが多い。トロントにも多数ある。カルガリーは規模は小さいが、充実している方だ。

| 内容 | 価格目安（CAD） |
|---|---|
| ラーメン・うどん | 16〜28 |
| 回転寿司・手頃な寿司 | 25〜50 |
| 居酒屋ディナー（1人・飲み物込み） | 45〜80 |
| 本格和食 | 80〜200 |

## 住居費

カナダの住居費は2020年代に入って急上昇した。特にトロントとバンクーバーは、北米の中でも高額な都市として知られている。

### 1ベッドルーム（1人〜カップル）

| 都市 | 月額目安（CAD） |
|---|---|
| トロント（市内） | 2,200〜3,800 |
| バンクーバー（市内） | 2,500〜4,000 |
| カルガリー | 1,500〜2,500 |
| モントリオール | 1,400〜2,400 |

### 2〜3ベッドルーム（家族向け）

| 都市 | 月額目安（CAD） |
|---|---|
| トロント（市内） | 3,500〜6,000 |
| バンクーバー（市内） | 3,800〜6,500 |
| カルガリー | 2,200〜4,000 |

### 郊外vs市内

トロントはMississauga・Markham、バンクーバーはBurnaby・Surrey・Richmondなどの郊外エリアに出ると家賃が20〜30%下がる。子育て環境・日本人コミュニティの状況も含めて判断する。

## 交通費

### 公共交通（TTC / TransLink / CTrain）

| 都市 | 主な交通機関 | 月定期（CAD） |
|---|---|---|
| トロント | TTC（地下鉄・バス） | 約158 |
| バンクーバー | TransLink（SkyTrain・バス） | 120〜175（ゾーンによる） |
| カルガリー | CTrain（LRT）+ バス | 112 |

バンクーバーのTransLinkはゾーン制を廃止（2020年）しており、定期代がシンプルになっている。

### 自家用車

カルガリー・郊外エリアは車がないと不便なケースが多い。

- ガソリン: CAD 1.4〜2.0/リットル（都市・時期により変動）
- 自動車保険: 月 CAD 100〜300（州・年齢・車種で大きく異なる。BC州はICBCが独占）

## 通信費

カナダの通信費は先進国の中で高い水準にある。

| キャリア | プラン例 | 月額（CAD） |
|---|---|---|
| Rogers | 20GB / 5G | 60〜100 |
| Bell | 20GB / 5G | 65〜110 |
| Telus | 20GB / 5G | 65〜105 |
| Freedom Mobile（サブブランド） | 20GB | 35〜55 |
| Fido / Koodo（サブブランド） | 10〜20GB | 40〜65 |

サブブランド（Fido / Koodo / Lucky Mobile等）はメインブランドより安いが、カバレッジが劣る場合がある。都市部ならサブブランドで十分という声も多い。

## 月額生活費の目安

### 一人暮らし（トロント想定）

| 項目 | 低め（CAD） | 高め（CAD） |
|---|---|---|
| 家賃 | 2,200 | 3,500 |
| 食費（自炊中心） | 300 | 600 |
| 交通費 | 158 | 250 |
| 通信費 | 40 | 100 |
| その他（日用品・外食・娯楽） | 300 | 800 |
| **合計** | **2,998** | **5,250** |

一人暮らしはCAD 3,000〜5,000/月（約33〜55万円）が目安。バンクーバーはトロントと同水準か若干高く、カルガリーはCAD 2,500〜4,000程度。

### 家族4人（子ども2人）

| 想定 | 月額目安（CAD） |
|---|---|
| トロント | 7,000〜12,000 |
| バンクーバー | 7,500〜13,000 |
| カルガリー | 5,000〜9,000 |

保育費（デイケア）は月 CAD 1,000〜2,000+になるケースが多い。連邦・州政府の子育て支援（CCB: Canada Child Benefit、$10-a-day childcare等）の活用でコストを抑えられる場合もある。
