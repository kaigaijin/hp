**この記事の日本円換算は、1AUD≒100円で計算しています（2026年4月時点）。為替は変動するので、現地通貨（AUD）の金額を基準にしてください。**

11月の第1火曜日、メルボルンのオフィスは空になる。しかしシドニーは普通に働いている。

メルボルンカップ・デー。ビクトリア州の一部（メルボルン都市圏）では祝日扱いになる競馬の日だ。「レースが止める国」とオーストラリア人は自称するが、正確には「レースが止める州」だ。オーストラリアの祝日制度は、州ごとに異なる。

## 連邦共通の祝日

全州で共通の祝日は以下の8日だ。

| 祝日 | 日付 |
|------|------|
| 元日（New Year's Day） | 1月1日 |
| オーストラリア・デー | 1月26日 |
| グッドフライデー | 復活祭前の金曜（年により変動） |
| イースター・サタデー | グッドフライデーの翌日 |
| イースター・マンデー | グッドフライデーの翌々日の月曜 |
| アンザック・デー | 4月25日 |
| 国王誕生日 | 州によって日付が異なる |
| クリスマス・デー | 12月25日 |
| ボクシング・デー | 12月26日 |

この中で「国王誕生日（King's Birthday）」がすでに州ごとに違う。ほとんどの州は6月の第2月曜日だが、西オーストラリア州は9月の第4月曜日、クイーンズランド州は10月の最終月曜日に設定されている。同じ祝日なのに、3〜4ヶ月ずれている。

ちなみに、国王チャールズ3世の実際の誕生日は11月14日だが、どの州もその日を祝日にしていない。

## 州独自の祝日

各州が独自に設定している祝日がある。例:

**ビクトリア州:** メルボルンカップ・デー（11月第1火曜日、メルボルン都市圏のみ）、AFLグランドファイナルの前日金曜日

**ACT（首都特別地域）:** リコンシリエーション・デー（5月27日、先住民との和解を記念）、ファミリー＆コミュニティ・デー

**南オーストラリア州:** プロクラメーション・デー（12月26日、ボクシング・デーの代わり）

**タスマニア州:** レクリエーション・デー（11月第1月曜日、北部のみ）

このため、「今日は祝日？」という質問への答えが州によって変わる。NSW州で働いている人がビクトリア州のクライアントに連絡したら「今日は祝日です」と自動返信が来る——ということが実際に起きる。

## 祝日の給与

祝日に働く場合の賃金は、Fair Work Act（公正労働法）で規定されている。

フルタイム・パートタイムの場合、祝日出勤は通常の2.5倍（250%）の割増賃金が適用される。カジュアル（パート・アルバイトに近い雇用形態）の場合は2.75倍（275%）。

具体例: 時給25AUD（約2,500円）のカジュアルスタッフがクリスマスに8時間働くと、25 x 2.75 x 8 = 550AUD（約55,000円）。1日で5万円以上。小売業やホスピタリティで祝日シフトが「稼ぎどき」と言われるのはこの計算による。

## 在住日本人への影響

日本の祝日は全国一律（年間16日）なので、オーストラリアの「州ごとに違う祝日」は最初戸惑う。

実務上の影響:

**給与計算。** 複数の州に拠点がある会社は、州ごとに祝日の割増賃金を計算する必要がある。会計ソフト（Xero、MYOB等）は州設定で自動対応するが、手計算だとミスしやすい。

**取引先との調整。** NSW州とビクトリア州にまたがる取引では、祝日のカレンダーが違うことを念頭に置く必要がある。

**学校の休み。** スクールホリデー（学期休み）も州ごとに1〜2週間ずれる。子どもの休みに合わせて旅行を計画する際は、自分の州のカレンダーを確認する必要がある。

**注意点:** オーストラリア・デー（1月26日）は近年「インベイジョン・デー（侵略の日）」と呼んで日付変更を求める運動が活発化している。先住民にとっては植民の開始を祝う日であり、政治的にセンシティブなテーマだ。職場では話題にする際に配慮が必要な場合がある。

祝日ひとつとっても、州の権限が強いオーストラリアの連邦制が見えてくる。「同じ国なのに違う」を、日常の中で体感する場面のひとつだ。
